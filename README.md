# 📘 LeetCode DSA Solutions

A curated collection of **LeetCode problems** solved using C++ with a focus on building strong Data Structures and Algorithms fundamentals.

---

## 🚀 Overview

This repository contains solutions to a wide range of LeetCode problems.  
Each problem is organized using its **problem number and title**, making it easy to search and navigate.

---

## 🗂️ Structure

All problems are stored in the root directory in the following format:

```bash
<problem-number>-<problem-name>

13-roman-to-integer
121-best-time-to-buy-and-sell-stock
141-linked-list-cycle
189-rotate-array
1013-fibonacci-number

```

## 🗂️ What Each Problem Includes

Each problem folder/file typically includes:

- ✅ Solution code (C++)  
- ✅ Optimized approach  
- ✅ Clean and readable implementation
